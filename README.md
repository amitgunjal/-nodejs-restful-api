## **Restful web services using Nodejs, Express, and MySQL**

 - Prerequisite 
	 - Nodejs
	 - MySQL
 - Database Setup
	 - source `app/migrations/queries.sql`
 - Create `.env` at root level
	 - Introduce `PORT`, `HOST`, `USER`, `PASS`, and `DB`
 - How to Run
	 - `yarn && yarn start`
 - Contribution
	 - This project is open for contribution
