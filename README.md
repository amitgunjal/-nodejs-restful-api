### Prerequisites

- Node JS
- MySQL
- Express JS

## Deployment

Create `.env` file.
```env
PORT=3000
HOST=localhost
USER=amit
PASS=root
DB=nodejs_contacts_db
```

## Deployment
```shell
$ yarn && yarn start
```
